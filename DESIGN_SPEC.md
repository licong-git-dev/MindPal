## 1. 设计概述

### 1.1 设计理念

**"未来的温度" - 科技感×温暖陪伴的完美融合**

MindPal的设计将冷静科技美学与情感化设计深度融合，创造一个既有未来感又充满人情味的数字人平台。我们借鉴Replika的情感化交互、Character.AI的对话流畅性和Soul的温馨治愈感，但通过3D写实数字人和沉浸式体验超越它们。

### 1.2 目标用户

- **单身白领 (18-35岁)**: 需要简洁现代、情感共鸣的界面
- **学生群体 (15-25岁)**: 需要活力、清晰的知识服务UI
- **老年群体 (60+岁)**: 需要大字体、高对比度、简单直观的界面

### 1.3 设计原则

1. **情感优先（Emotion First）**
   数字人的每个微表情、每次呼吸动画都传递温度，色彩、动效、文案都围绕"陪伴感"设计

2. **科技赋能（Tech-Enabled）**
   玻璃拟态设计营造未来感（半透明卡片、柔和阴影、光线高光），流畅的淡入淡出过渡，细腻的微动效

3. **极简直观（Minimalist & Intuitive）**
   去除冗余，让用户专注于与数字人的对话，熟悉的图标语言，老年用户也能轻松上手

4. **沉浸体验（Immersive Experience）**
   主对话页数字人占据视觉中心（60%屏幕），对话气泡半透明悬浮，不遮挡数字人表情

### 1.4 设计关键词

`科技感` `温暖陪伴` `玻璃拟态` `流畅动效` `情感化` `沉浸式` `3D写实` `渐变紫蓝`

#### 交互逻辑

**流程1: 文字对话**
1. 用户输入 "今天好累" → 输入框聚焦，边框变蓝
2. 点击发送 → 消息气泡从右侧滑入（200ms），输入框清空
3. 数字人头像显示"思考中"动画 → 加载圈旋转（P0）或 眨眼动画（P1）
4. 系统调用情感识别 → 检测到"疲惫"情绪
5. 情绪指示器切换为"难过"😢 → 缩放弹性动画
6. 数字人回复出现 → 左侧气泡淡入 + 逐字打印效果（50ms/字）
7. 用户可以继续输入...

**流程2: 语音对话**
1. 用户长按🎤按钮 → 按钮背景变红，开始录音
2. 录音界面覆盖输入区 → 显示波形动画 + "松手发送，上滑取消"提示
3. 用户说话 → 波形动画跟随音量变化
4. 松手发送 → 波形消失，显示"识别中..."加载动画
5. ASR识别完成 → 文字"今天工作好累啊"显示在右侧气泡
6. 同时自动发送给数字人 → 后续流程与文字对话一致
7. 数字人语音回复（TTS） → 左侧气泡出现 + 🔊播放图标
8. 点击🔊重新播放语音

**流程3: 查看对话历史**
1. 页面加载时自动请求最近50条消息
2. 骨架屏显示加载状态 → 3个气泡轮廓
3. 消息加载完成 → 气泡从上到下依次淡入（每个间隔50ms）
4. 自动滚动到最新消息
5. 用户向上滚动查看更早消息 → 触底时加载更多（分页）

### 7.3 实现注意事项

**性能优化**:
1. 图片懒加载: 使用 `loading="lazy"` 属性
2. 数字人头像: 预加载关键图片，`<link rel="preload">`
3. 动画性能: 优先使用 `transform` 和 `opacity`，避免 `width`/`height` 动画
4. 玻璃拟态: `backdrop-filter` 性能消耗较大，避免过度使用

**可访问性（A11y）**:
1. 颜色对比度: 文字与背景对比度 ≥ 4.5:1（WCAG AA标准）
2. 键盘导航: 所有交互元素可通过Tab键访问
3. 语义化HTML: 使用 `<button>`, `<nav>`, `<main>` 等语义标签
4. ARIA标签: 为图标按钮添加 `aria-label`
5. 焦点可见: 明确的焦点样式（`:focus-visible`）

**浏览器兼容性**:
1. 玻璃拟态: `backdrop-filter` 需添加 `-webkit-` 前缀
2. CSS变量: IE11不支持，使用PostCSS插件降级
3. 圆角: `border-radius: 9999px` 在所有现代浏览器支持
4. 渐变: 使用标准语法 + `-webkit-` 前缀

**深色/浅色模式切换**:
```css
/* 使用CSS变量 + 类名切换 */
:root {
  --color-bg: #0F172A; /* 深色模式默认 */
  --color-text: #F1F5F9;
}

.light-mode {
  --color-bg: #F8FAFC;
  --color-text: #0F172A;
}
```

**字体加载优化**:
```html
<!-- 预加载关键字体 -->
<link rel="preload" href="/fonts/Inter-Regular.woff2" as="font" type="font/woff2" crossorigin>
```

**动画性能**:
```css
/* 使用will-change提示浏览器优化 */
.digital-human-avatar {
  will-change: transform;
  animation: breathing 2s ease-in-out infinite;
}

/* 动画完成后移除will-change */
.digital-human-avatar:not(:hover) {
  will-change: auto;
```
