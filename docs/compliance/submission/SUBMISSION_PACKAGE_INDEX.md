## 命名规范

- 所有 PDF 以 `NN_N_XX.pdf` 命名（分类号_序号_内容）
- 中文文件名直接用，网信办系统接受 UTF-8
- 每份 PDF 确保可搜索（文字层 OCR，不要纯图扫描）
- 盖章页用"带章骑缝"方式更正式

## 大小控制

- 单个 PDF ≤ 20 MB（过大系统可能拒绝）
- 总包 ≤ 500 MB（用 zip 压缩）
- 截图导出用 PNG，控制在 1920×1080 以内

## 提交渠道

### 全国互联网信息服务算法备案系统

- 地址：[https://beian.cac.gov.cn/](https://beian.cac.gov.cn/)
- 登录：法人或授权人账号
- 流程：
  1. 选择"深度合成服务算法备案"
  2. 填写在线表单
  3. 附件上传（每个 < 20 MB）
  4. 法人签字声明（线上或线下）
  5. 提交后等待网信办分派审核员

### 生成式 AI 服务备案

- 同一平台，选择"生成式人工智能服务备案"
- 建议**和深度合成备案同时提交**（大部分材料可复用）

### 线下补材料（如要求）

- 按审核员指示邮寄实体材料到所在地网信办

## 提交后跟进

### 状态查询

备案号下来前，定期登录平台查看状态：

- `待受理` → 已提交但未分配审核员
- `审核中` → 正在审
- `需补正` → 点进去看具体缺什么
- `已通过` → 备案号公布

### 补正响应

- 7 天内补齐（逾期重新走流程）
- 补正内容**必须针对原反馈**，不要夹带新内容

### 拿到备案号后

1. **立即**更新 frontend/legal.html 和所有页面 footer 的占位符为真实备案号
2. **立即**在服务首次打开时展示备案号（通常是第一个界面底部）
3. 发公告：《关于 MindPal 完成 XX 备案的说明》
4. 整理提交经验到 `docs/compliance/LESSONS_LEARNED.md` 供将来参考

## 备案号公示位置

根据《生成式人工智能服务管理暂行办法》§17 要求，备案号必须公示在：

- [ ] 网站主页底部（footer）
- [ ] 登录页
- [ ] 用户服务协议
- [ ] 首次使用时向用户展示（可以是一个"知道了"按钮）
- [ ] 应用市场简介中（如上架小程序 / App）

## 变更备案

以下情况需要**重新提交变更备案**：

- 基础模型更换
- 核心功能重大变化
- 法定代表人变更
- 公司名称或经营范围变化
- 应用名称变化

变更周期比首次备案短（10-30 天）。
