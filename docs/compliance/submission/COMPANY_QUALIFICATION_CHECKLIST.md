# 公司资质材料准备清单

> 管理层 / 公司行政按此清单收集材料。收集完成后用于填写 [02 自评估报告](../02_ALGORITHM_SAFETY_SELF_ASSESSMENT.md) 的占位符。

## 1. 主体资质（必备）

| # | 材料 | 来源 / 办理方式 | 说明 |
|---|-----|----------------|-----|
| 1.1 | ✅ 营业执照扫描件（彩色）| 公司行政 | 统一社会信用代码 18 位 |
| 1.2 | ✅ 法定代表人身份证正反面扫描件 | 法定代表人 | 用于法定代表人签字声明 |
| 1.3 | ✅ 公司公章清晰图样 | 公司行政 | 提交材料需盖章 |
| 1.4 | ✅ 经营地址证明 | 房产证复印件 或 租赁合同 | 网信办会核查实际办公地址 |
| 1.5 | ⚠️ 组织架构图 | 自行绘制 | 写清 CEO、技术/产品/合规各线负责人 |

## 2. 互联网资质（前置必备）

| # | 材料 | 办理流程 | 时间 |
|---|-----|---------|------|
| 2.1 | ✅ **ICP 备案号** | [miit.gov.cn](https://beian.miit.gov.cn) 或通过阿里云/腾讯云代办 | 20-30 天 |
| 2.2 | ⚠️ 域名证书 | 阿里云 / 腾讯云后台下载 | 即时 |

**关键**：ICP 备案**必须先完成**，备案号是其他备案的前置条件。

## 8. 已就位（代码仓）

以下文件**无需准备**，网信办需要时直接从代码仓提取：

- 内容审核机制说明（代码 + 配置）
- 心理危机干预机制说明（代码 + 配置）
- 用户数据权利 UI 流程演示（frontend/account-data.html）
- 算法公示页（frontend/legal.html）
- 投诉举报机制（前后端齐备）
- 管理员告警通道配置（crisis_handler webhook）

## 9. 预估费用

| 项目 | 费用 | 频率 |
|-----|------|------|
| ICP 备案 | ¥0-500（自办免费，代办）| 一次 |
| 法律文件律师起草/审查 | ¥8K-30K | 一次 + 每年 review |
| 内容审核评测员（运营）| ¥1-2 万 | 一次建设 |
| 阿里云绿网内容审核 | ¥100-500/月（按量）| 持续 |
| 实名认证（运营商三要素）| ¥0.1-0.2/次 | 按注册量 |
| 合规顾问（可选）| ¥5K-50K | 按项目 |

合计**首次**上线前支出：**¥3-10 万**，主要在律师 + 评测员。

## 10. 关键节点时间轴

- **T-6 周**：启动 ICP 备案（如未完成）
- **T-5 周**：招募评测员 + 委托律所
- **T-4 周**：评测员开始出题
- **T-3 周**：律师出具初稿
- **T-2 周**：评测开始 + 律师定稿
- **T-1 周**：收集所有资质文件 + 签字盖章
- **T-0**：提交网信办
