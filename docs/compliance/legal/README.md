## 如何使用

1. **内部评审**：由创始人 / 产品 / 合规同学先读一遍确认业务表达准确
2. **委托律所**：找有"互联网+AI"经验的律所（上海/北京/杭州常见）
   - 预算参考：¥8,000 - ¥30,000（3 份文件打包）
   - 周期：2-3 周
3. **定稿后**：
   - 放到 `frontend/legal.html` 替换占位内容
   - 同步放到 `docs/compliance/legal/` 作为仓库留档
   - 在产品首次打开时弹出提示用户阅读
4. **定期维护**：法规有重大更新时同步修订（建议每 6 个月 review 一次）

## 红线提醒

以下表达**不能**出现在法律文件里（会被监管认定虚假承诺）：

- ❌ "100% 安全" / "绝对保密" / "永不泄漏"
- ❌ "不收集任何个人信息" （与实际不符）
- ❌ "AI 判断准确" / "可替代专业医生"
- ❌ "一旦订阅不予退款" （违反《消费者权益保护法》）
- ❌ "我们有权随时修改协议无需通知" （违反公平原则）

## 必须包含的必选项（PIPL §17 告知义务）

隐私政策**至少**包含:

1. 处理者名称 + 联系方式
2. 处理目的 + 方式
3. 信息种类 + 保留期限
4. 用户行使权利的方式 + 程序
5. 未成年用户特别保护
6. 共享 / 委托处理 / 对外提供的情况
7. 境外传输（如有）的情形
8. 出境安全评估情况（如有）
9. 变更历史
