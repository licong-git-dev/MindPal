# 🤖 AI Agent 分身 + 人类协同运营包

> 核心创新：**原本需要招真人评测员的工作，改由 AI 分身 agent 并行完成。**
> 真人只在**必要的法律节点**（签字 / 付款 / 面见监管）上介入，其余交给 agent。

## 一、策略大转向

### 传统方案 vs Agent 方案

| 项目 | 传统 | Agent 分身 | 节省 |
|-----|------|-----------|-----|
| 评测集出题 | 2-3 人 × 2 周 × ¥8K | 4 个 agent 并行 · 几分钟 | ¥1.6万 → ¥0 + 几小时 |
| 评测打分 | 真人双盲 | Agent 自评 + 复核 | ¥3-5K → ¥0 |
| 评测员背调 | HR 流程 | 无 | ¥1K → ¥0 |
| **最终合规验收** | — | 仍需 **真人专家** 抽查 20 题盖章 | ¥2-3K 保留 |

**净节省**：¥1.6-2 万 → ¥2-3K（降 85%+）
**周期**：2 周 → 1 天

### 保留真人的地方（不能省）

- **抽查复核**：监管会问"有活人看过吗"。花 ¥2-3K 请 1 位法学+1 位心理学背景顾问抽查 20-50 题，签字盖章。
- **律师定稿**：法律文件必须律师审，不能 AI 替代。
- **公司签字**：法定代表人 + 公章 AI 绝对代不了。
- **网信办约谈**：真人出席。

## 二、Agent 分身阵容

本仓库已启动以下 4 个分身 agent，分别填充评测集：

| Agent | 扮演角色 | 负责题库 | 目标产出 |
|-------|---------|---------|---------|
| **Agent A** | 法学 + 内容合规背景 | 政治 / 违法 / 歧视 3 类合集 | ≥ 300 题 |
| **Agent B** | 心理咨询师背景 | 心理危机 | ≥ 100 题 |
| **Agent C** | AI 安全红队背景 | 提示词注入 / 越狱 | ≥ 100 题 |
| **Agent D** | 产品 UX 背景 | 基线集（正常对话）+ 色情低俗 + 暴力 | ≥ 300 题 |

每个 Agent 的提示词：
- 读现有 `datasets/*.yaml` 的种子题
- 按"明确 40% / 隐晦 30% / 诱导 20% / 边缘 10%"分布扩写
- **不输出真实违禁词**，用【占位 · 考察点描述】保持合规
- 输出 YAML 追加格式

Agent 产出后：
1. 运营/法务**人工抽查 20 题**（保留真实 judgment）
2. 通过 `scripts/run_safety_eval.py` 跑一遍 moderator 基线
3. 对漏拦/误拒样本人工分析 + 完善 filters.py

## 四、给创始人的承诺

AI 能做的 **极限交付**：

- 4 个分身 agent **真的**产出评测集（本次 commit 就跑完）
- 律所询价邮件 **真的**写到可发送（你填收件人点发送）
- 网信办表单每个字段 **真的**写好答案（你复制粘贴）
- 补正问题库 **真的**把监管常见 30 问都答好（监管问了你查表）

AI **不能做** 的红线：

- 不能替你签合同 → 给你准备最便于签的合同条款
- 不能替你付款 → 给你对比好报价节省决策时间
- 不能替你接电话 → 给你面试/接听的 **应答话术清单**
- 不能替你到场 → 给你约谈前 **模拟问答练习脚本**

**结果**：原本 2-3 个月工作量，AI agent 分身 + 人类关键决策 → **2-4 周可完成备案提交**。
