# 平台注册 + 实名认证

> 2026 年 4 月为准的操作流程。如官网有变化请以实际页面为准。
> 过程中遇到任何报错，**截图**保存便于后续申诉。

## 一、系统入口

**全国互联网信息服务算法备案系统**

- URL: https://beian.cac.gov.cn/
- 入口位置：首页顶部「**登录**」或「**注册**」

## 二、账号注册

### Step 1 选择账号类型

- 🔘 **企业账号**（推荐 MindPal 用这个）
- ⚪ 个人账号（面向开发者不太适用）

### Step 2 填写注册信息

```
单位名称:       🟨 [公司全称，与营业执照完全一致]
统一社会信用代码: 🟨 [18 位]
法定代表人姓名:   🟨
法定代表人证件号: 🟨 [身份证]
法定代表人手机:   🟨
单位注册地址:     🟨 [与工商登记一致]
单位通讯地址:     🟨 [实际办公地址，可与注册地不同]
邮政编码:         🟨
单位电话:         🟨
```

### Step 3 填写管理员信息

**建议**：不用法定代表人做日常操作员，改指派合规负责人。

```
管理员姓名:  🟨 [合规负责人姓名]
管理员证件: 🟨 [身份证]
管理员手机: 🟨 [不要和法人同号]
管理员邮箱: 🟨
用户名:     🟨 (建议: mindpal_compliance)
密码:       🟨 (≥ 8 位，含大小写+数字+符号)
```

### Step 4 提交 → 等邮件激活

- 5-10 分钟内收到激活邮件
- 如 30 分钟没收到，检查垃圾邮件或联系系统客服（首页有电话）

## 三、实名认证

登录后**第一件事**是实名认证。

### Step 1 企业实名认证

点击账号头像 → **企业认证**

需上传：
- [ ] 营业执照照片（PDF 或 JPG，<5MB）
- [ ] 法定代表人身份证正反面（两张）
- [ ] 授权委托书（如管理员不是法定代表人）—— **模板系统提供**

### Step 2 授权委托书填写

如果**管理员 ≠ 法定代表人**（推荐），需要法定代表人签授权书：

```
授权委托书

今委托 🟨 [管理员姓名]（身份证号: 🟨）作为本单位备案工作的授权代表，
全权处理 🟨 [公司全称] 向互联网信息服务算法备案系统提交的所有备案
事项，包括但不限于：

1. 在系统中填写、修改、提交备案信息及附件；
2. 代表本单位接收系统通知、审核意见；
3. 在本单位授权范围内处理补正、修改、撤回等操作。

授权期限：自 🟨 YYYY-MM-DD 起至 🟨 YYYY-MM-DD 止（建议 1 年）。

委托单位：🟨 [公司全称]（盖公章）
法定代表人：🟨 [姓名]（签字）
日期：🟨 YYYY-MM-DD
```

打印 → 公章 + 法定代表人签字 → 扫描 PDF → 上传。

### Step 3 等审核

- 一般 **1-3 个工作日**通过
- 审核通过后才能开始填写备案表单

## 四、数字证书准备

网信办系统要求**最终提交时**用**数字证书签名**。

### 选项 1：使用电子签名（推荐，快）

- 系统内置"电子签章"功能
- 绑定法定代表人手机号 + 实名验证后可直接签
- 免费，适用于绝大多数场景

### 选项 2：CA 数字证书（慢）

- 向 CA 机构（中经认证 / 天威诚信 等）申请
- 费用 ¥300-800/年
- 7-10 天邮寄 UKEY
- 使用时插 UKEY 到电脑签名
- 仅在系统不接受电子签名时用

**MindPal 建议**：优先电子签名，不行再 CA。

## 五、登录后的重要页面

认证通过后，左侧菜单重点关注：

| 菜单 | 用途 |
|-----|------|
| 「**算法备案**」 | 填新的备案申请 |
| 「**备案管理**」 | 查已提交的备案状态 |
| 「**补充补正**」 | 监管反馈后的补正入口 |
| 「**系统消息**」 | 监管审核员的沟通 |
| 「**帮助中心**」 | 遇到问题先来这里 |

**强烈建议**：
- 管理员手机**绑定系统通知**（短信+邮件），监管反馈时立即知道
- 管理员**每日登录一次**看有没有新消息

## 六、常见问题

### Q1 注册时提示"统一社会信用代码已存在"

- 说明贵公司之前注册过
- 找之前的管理员密码，或点"**忘记密码**"

### Q2 法人身份证上传后一直审核中

- 可能图片不清晰
- 重拍（白底、四角全、文字清晰）
- 单张 < 5MB 但要 > 100KB（太小也不行）

### Q3 管理员手机 / 邮箱忘了

- 拿营业执照去网信办窗口申诉（较慢）
- 或拨打系统客服（官网底部）

### Q4 授权委托书被拒

- 90% 是没有**公章**或**签字**
- 不能用电子签名的授权书，必须**实体签字 + 实体公章**再扫描
