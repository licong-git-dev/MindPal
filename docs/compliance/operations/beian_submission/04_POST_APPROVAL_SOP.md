# 拿到备案号后的 7 步上线 SOP

> 恭喜！备案号下来是好消息，但**不是终点**，而是**公开运营的开始**。
> 按下列 7 步完成才算真正合规上线。

## Step 1 · 立即截图留证

**时间**：拿到备案号当日

- [ ] 登录网信办系统，到"备案管理"页面
- [ ] 截图"备案详情"（含备案号、批复日期、有效期）
- [ ] 保存到 `docs/compliance/operations/beian_submission/approved/` 目录
- [ ] **三份备份**：企业云盘、个人电脑、打印一份存档

**为什么**：万一将来有争议，系统可能变更，自己存档最稳。

## Step 2 · 前端公示（48 小时内）

**时间**：拿到备案号后 2 天内

替换代码里所有 `🟨 网信算备[编号]` 占位符为真实备案号：

```bash
cd /home/yake/MindPal
# 批量替换（grep 先看有哪些地方）
grep -rn "网信算备\[编号\]" frontend/ docs/
grep -rn "网信算备 \[编号\]" frontend/ docs/

# 确认无误后手动替换（或用 sed 批量，小心符号）
```

**必须展示备案号的位置**（监管必查）：

- [ ] `frontend/index.html` footer
- [ ] `frontend/dh-list.html` footer
- [ ] `frontend/legal.html` footer + 算法公示 tab
- [ ] 所有法律文件（服务协议 / 隐私政策 / 未成年声明）
- [ ] 应用市场简介（如上架小程序 / App）
- [ ] **首次使用弹窗**（用户第一次打开服务时看到）

## Step 3 · 公开发布"已完成备案"通告

**时间**：拿到备案号后 3-5 天

### 站内公告

在 `frontend/dh-list.html` 或首页加一条**置顶公告**：

```
🎉 MindPal 已完成算法备案

尊敬的用户:

MindPal 已于 YYYY-MM-DD 完成国家网信办的
- 互联网信息服务深度合成算法备案 (网信算备 XXX)
- 生成式人工智能服务备案 (XXX)

这是我们向合规、负责任运营迈出的重要一步。感谢您的等待。

— MindPal 团队
```

### 对外渠道同步

- [ ] 公司微信公众号
- [ ] 小红书 / 抖音品牌账号（如有）
- [ ] 相关行业媒体（36Kr / 虎嗅等，可选）

## Step 4 · 解除内测白名单

**时间**：Step 2 完成后

如在备案期间开启了内测白名单限流（限 500 用户内），现在可以：

- [ ] 在 feature flag 里关闭白名单检查
- [ ] 开放公开注册
- [ ] 应用商店提交上架（如未上架）

## Step 5 · 投流准备

**时间**：Step 4 完成后

- [ ] 小红书 / 抖音 / B 站等投放渠道的材料准备
- [ ] 投放素材中**必须显示备案号**
- [ ] 落地页 footer 更新备案号
- [ ] 咨询 DEEP_AUDIT_V3 的 Lane B 种子发布策略

## Step 6 · 持续合规运营（30 天）

拿到备案号后第 1 个月是"**合规摩擦期**"，容易出问题：

- [ ] 每日查看系统消息
- [ ] 每日人工复核 100 条对话样本
- [ ] 首个月主动**抽样用户投诉 + 反馈**
- [ ] 发现任何监管风险**立即**上报合规负责人
- [ ] 每周把以下指标写到 WEEKLY_REPORT：
  - 日活 / 新注册 / 活跃用户对话数
  - 内容审核拦截次数 / 类别分布
  - 心理危机事件数 / 处置结果
  - 用户举报数 / 处理时延
  - 支付转化 / 退费 / 客诉

## Step 7 · 建立长期合规机制

**时间**：拿到备案号后 1-3 个月

- [ ] **每季度自查一次**备案信息时效性
- [ ] **每半年** review 法律文件是否需更新
- [ ] **每年**全流程应急演练 1 次
- [ ] 订阅合规咨询（¥2-5K/月，值得）
- [ ] 建立**合规日志**：每次法规变更 / 重大事件都记录

## 触发变更备案的情况

以下情况发生后必须主动申请变更备案（不等监管来问）：

| 触发事件 | 变更时限 |
|---------|---------|
| 基础模型更换（如从通义千问换到百度文心）| 立即变更 |
| 核心功能重大调整（如新增 3D 虚拟场景）| 上线前变更 |
| 法定代表人变更 | 工商变更 30 天内 |
| 公司名称变更 | 工商变更 30 天内 |
| 服务对象扩展（如新增 ToB 线）| 上线前变更 |

变更备案比首次备案快（一般 2-4 周）。

## 应对舆情突发

拿到备案号后如遇**舆情事件**（负面报道 / 用户集体投诉 / 媒体质疑）：

- [ ] 立即**暂停**相关功能（用 feature flag）
- [ ] 1 小时内 CEO + 法务 + 合规 讨论出**统一口径**
- [ ] 24 小时内向网信办主动报告（不等监管来问）
- [ ] 按照 [INCIDENT_RESPONSE_SLA](../../submission/INCIDENT_RESPONSE_SLA.md) P0 流程处置

**宁愿主动报告显得小题大做，也不要等监管来查被动挨打**。
