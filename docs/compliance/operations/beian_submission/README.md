## 基本信息

**系统入口**：[全国互联网信息服务算法备案系统](https://beian.cac.gov.cn/)

**需要准备**（按 [COMPANY_QUALIFICATION_CHECKLIST.md](../../submission/COMPANY_QUALIFICATION_CHECKLIST.md) 全部就位）:

- 公司数字证书（可用电子签名替代）
- 法定代表人身份证
- ICP 备案号
- 营业执照
- 所有附件 PDF（按 [SUBMISSION_PACKAGE_INDEX.md](../../submission/SUBMISSION_PACKAGE_INDEX.md) 准备）

**两个备案分别提交**：

- 深度合成算法备案（30-60 天）
- 生成式 AI 服务备案（60-120 天）

**建议**：同一天提交两个，大部分材料可复用，节省时间。
