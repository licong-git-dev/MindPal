# 📊 项目跟踪包 · 从今天到备案号到手

> 4 份文档组成一个轻量级项目管理系统，**不用企业微信不用 Notion**，
> 仅靠 markdown 就能跟踪"备案提交"整个项目。

## 文档清单

| 文件 | 用途 | 更新频率 |
|-----|------|---------|
| [TIMELINE.md](TIMELINE.md) | 从 T-4 周到 T+12 周的甘特图 | 一次性制定，月度 review |
| [KANBAN.md](KANBAN.md) | 按"To Do / Doing / Done"展示所有任务 | 每日 update |
| [WEEKLY_REPORT_TEMPLATE.md](WEEKLY_REPORT_TEMPLATE.md) | 周报模板 | 每周五填 |
| [MILESTONES.md](MILESTONES.md) | 关键里程碑 checklist | 达成时打勾 |

## 用法

1. **开工第一天**：
   - 按 [TIMELINE.md](TIMELINE.md) 填自己的真实日期
   - 把 [KANBAN.md](KANBAN.md) 的"To Do"列填满你的所有任务
2. **每天**：
   - 更新 KANBAN 上的 task 状态
   - 遇阻写进 `/blockers` 区块
3. **每周五 17:00**：
   - 5 分钟写周报（基于 WEEKLY_REPORT_TEMPLATE）
   - 发给共同负责人 + 自己归档
4. **到达里程碑**：
   - 在 MILESTONES 打勾 + 记录实际完成日期
   - 立即开始下一个里程碑

## 心法

- **不求完美**，求**更新**
- **文档即事实**，口头承诺不算
- **阻塞 48 小时必须升级**（自己解决不了就找人）
