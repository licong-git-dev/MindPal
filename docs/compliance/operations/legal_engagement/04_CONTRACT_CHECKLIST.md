# 委托合同 · 关键条款检查清单

> 收到律所合同草案后，按此清单逐项核查再签字。
> 每项**必查**，不能只是信任律所"标准合同"。

## 必备条款（缺一不签）

### 1. 服务范围 — 精确描述

- [ ] 明确列出**交付物清单**（3 份文件名 + 字数）
- [ ] 明确包含**修订次数**（至少 2 轮）
- [ ] 明确是否包含**合规意见书**
- [ ] 明确是否包含**备案提交后监管反馈应答**
  - ⚠️ 如不含，明确后续按小时费率是多少

### 2. 交付期限 — 可追责

- [ ] 初稿交付时间：签约后 🟨 [N] 日内
- [ ] 每轮修订响应：🟨 [N] 个工作日内
- [ ] 最终定稿：签约后 🟨 [N] 日内（建议 30 日）
- [ ] 延期责任：律所延期超 🟨 [N] 天按日罚 🟨 [X]%

### 3. 费用与付款 — 防止模糊

- [ ] 总价**明确**（包含增值税否？）
- [ ] 付款节奏：建议 **50% 签约 + 50% 定稿**（不要全款先付）
- [ ] 额外费用触发条件（增加文件 / 监管补正 / 超出修订轮次）**逐项明细**
- [ ] 差旅费：约定"北京/上海内不另收，出差按实报"
- [ ] 发票类型：增值税专用发票（便于抵扣）

### 4. 知识产权 — 文件归属权

- [ ] 定稿文件的**著作权归我方**（律所仅保留署名权）
- [ ] 律所**不得**将本文件用于其他客户
- [ ] 本合同结束后律所保留文件备份需经我方同意

### 5. 保密 — 双向

- [ ] 律所承担保密义务
- [ ] 但我方要求"经我方同意可作为**行业案例**匿名引用"（不禁止）
- [ ] 保密期限：合同结束后 🟨 [3-5] 年
- [ ] 违约责任：明确违约金数额

### 6. 质量保证与验收

- [ ] **验收标准**：是否通过网信办备案审核？这个标准太高，不合理
  - 建议改为："依据 XX 法规起草，无明显合规遗漏"
- [ ] **验收方式**：我方收到定稿后 🟨 [7] 日内书面反馈是否验收
- [ ] **瑕疵处理**：若后续发现文件存在**明显法律错误**，律所**免费**修改

### 7. 责任与限制

- [ ] 律所**法律意见的最大赔偿上限** = 合同总价的 🟨 [1-3] 倍
- [ ] 排除条款：不对监管**结果**承担责任（监管不通过 ≠ 律所有过错）
- [ ] 第三方责任除外：如基础模型提供方政策变化导致文件需修改，不视为律所违约

### 8. 合同终止

- [ ] 任一方可**提前 30 日书面**终止
- [ ] 律所提前终止，退还未履行部分费用 + **赔偿我方 🟨 [10]% 违约金**
- [ ] 我方提前终止，律所结算已完成工作（按天数或里程碑）

### 9. 争议解决

- [ ] 适用中华人民共和国法律
- [ ] **仲裁**还是**诉讼**？（建议仲裁 — 北京/上海国际仲裁委员会，更快）
- [ ] 管辖地：我方注册地（降低我方维权成本）

### 10. 通知与送达

- [ ] 双方通讯地址 + 邮箱 + 手机明确
- [ ] 电子邮件构成有效送达（便于快速沟通）

## 需要警惕的"霸王"条款

| 原文 | 问题 | 应改为 |
|------|------|-------|
| "本所承担的责任仅限于书面文件，不承担任何实际损失" | 免责过宽 | 限制为"合同金额的 N 倍" |
| "本所保留对本合同的最终解释权" | 单方解释权无效 | 删除或改为"双方协商一致" |
| "客户应提供一切本所要求的资料" | 义务无边界 | 改为"客户应配合提供合理必要的资料" |
| "本所可因客观原因延期交付，不承担责任" | 免责宽 | 列明**具体**的客观原因（不可抗力）|
| "本合同终止后，本所有权保留客户所有资料" | 数据权限混淆 | 要求"合同终止后 🟨 [30] 日内销毁或返还" |
| "客户同意接受本所其他客户的同行介绍" | 隐私风险 | 删除 |

## 附加条款建议（对我方有利）

这些默认不在律所合同里，但可要求加上：

1. **响应时效承诺**：邮件 24 小时回复，紧急事件 1 小时回复
2. **合伙人参与承诺**：保证合伙人**亲自把关**核心交付（不能全程甩给助理）
3. **6 个月合规咨询保障**：定稿后 6 个月内若相关法规有重大修订，律所**免费**出具修改建议
4. **避免利益冲突**：律所**不得**同时为我方主要竞争对手提供同类服务（至少合同期 + 6 个月）
5. **数据处理协议 (DPA)**：如果律所在服务中需要处理我方用户数据样本，单独签订 DPA

## 签字前最后 3 问

1. **问律所**："本合同有没有针对我行业的特殊条款？"（试探他们是否真的熟悉 AIGC）
2. **问律所**："最近 3 个月有没有类似项目合同我能看看（脱敏）"
3. **问自己**："如果监管 3 个月后拒绝备案，我能拿律所怎么办？"（答不上 → 验收标准要重谈）

## 签字流程

```
合同草案 → 我方内部 review 3 日 → 与律所谈判修订 → 终稿
   ↓
我方盖公章 + 法定代表人签字（双份原件）
   ↓
律所盖公章 + 承办律师签字
   ↓
双方各执一份，扫描件邮件归档
   ↓
支付首付款 → 开始服务
```
