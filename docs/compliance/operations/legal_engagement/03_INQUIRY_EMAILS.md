## 邮件发送前的 5 秒检查

- [ ] 🟨 占位符**全部**替换了？（用 Ctrl-F 搜 "🟨" 应该 0 结果）
- [ ] 收件人填了 2-3 家不同律所？（BCC 不要用，一对一发）
- [ ] 附件文件名不含乱码？
- [ ] 附件加密了？（内有营业执照等敏感信息，PDF 建议设密码 + 另发密码）
- [ ] 自己读一遍邮件，有没有错别字 / 表达别扭？
- [ ] 发送时间是工作时间？（避免深夜发以免显得不专业）

## 收到回复后的判断

### 好的回复特征

- 回复周期 ≤ 48 小时
- 回答了 RFP 里**每个问题**（不是只挑好的答）
- 明确写**局限** / **假设** / **风险点**（体现专业性）
- 报价**具体到项**，不是"一口价 ¥XX"
- 主动提**补正/延伸条款**的费率

### 坏的回复特征

- > 5 天才回
- 只发一份通用律所简介，不针对 RFP
- "电话联系详细商谈"（逃避书面报价）
- 一口价 ¥XX 但不拆分
- 承诺"包过备案"（假的，监管不保证）
- 纠缠要求先签保密协议才报价（不合理）

## 发出后第 7 日如果没有合适回复

有两种补救：

1. **加一轮催办邮件**：
   ```
   您好，📧 之前发的询价邮件是否方便今明两天给予回复？我们需要
   在 🟨 [日期] 前确定合作律所。感谢！
   ```
2. **扩大范围**：再发 2-3 家**第二梯队**律所
3. **退一步**：考虑 C 类合规咨询公司降本
