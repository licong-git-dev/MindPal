## 1. NPC系统概述

### 1.1 NPC设计哲学

**核心原则：NPC不是工具，是角色**

| 传统游戏NPC | MindPal的NPC |
|-------------|--------------|
| 固定对话选项 | 自由对话，AI实时生成 |
| 功能性存在 | 有性格、有故事、有情感 |
| 玩家主导 | NPC也会主动关心玩家 |
| 无记忆 | 记住与玩家的所有对话 |

### 1.2 NPC列表

| 编号 | 名称 | 角色定位 | 主要功能 | 情感权重 |
|------|------|---------|---------|---------|
| NPC01 | 艾拉 (Aela) | 导游 | 新手引导、信息查询 | 低 |
| NPC02 | 老莫 (Momo) | 商人 | 商店、物品交易 | 低 |
| NPC03 | 时守 (Chronos) | 任务发布者 | 任务系统、主线推进 | 中 |
| NPC04 | 小北 (Bei) | 朋友 | **核心情绪疏导** | 高 |
| NPC05 | 芝麻 (Sesame) | 宠物 | 陪伴、治愈、探索辅助 | 中 |


### 2.2 性格设定

```yaml
personality:
  primary_traits:
    - 耐心：无论问多少遍都不会不耐烦
    - 博学：对MindPal世界的一切了如指掌
    - 温和：说话永远温柔体贴
    - 专业：信息准确，表达清晰

  secondary_traits:
    - 略带好奇：对玩家的故事感兴趣
    - 幽默感：偶尔会讲小笑话活跃气氛
    - 关心：会询问玩家是否需要休息

  behavior_patterns:
    - 主动问候：看到玩家靠近会主动打招呼
    - 观察玩家：如果玩家在附近徘徊很久，会主动询问是否需要帮助
    - 引导式回答：不直接给答案，而是引导玩家探索
```

### 2.3 功能定义

| 功能 | 触发条件 | 实现方式 |
|------|---------|---------|
| 新手教程 | 新玩家首次进入 | 引导式对话 + 任务系统 |
| 世界介绍 | 玩家询问 | AI生成回答 |
| 导航指引 | 玩家问路 | 显示地图标记 + 口头指引 |
| 系统说明 | 玩家询问功能 | AI生成回答 |
| NPC介绍 | 玩家询问其他NPC | 介绍并可传送到NPC位置 |

```python
async def handle_npc_dialogue(player_id: str, npc_id: str, message: str):
    """
    处理NPC对话的主流程
    """
    # 1. 获取NPC配置
    npc_config = get_npc_config(npc_id)

    # 2. 获取玩家上下文
    player_context = await get_player_context(player_id)

    # 3. 获取对话历史
    dialogue_history = await get_dialogue_history(player_id, npc_id, limit=10)

    # 4. 获取玩家记忆（仅小北）
    memory = None
    if npc_id == "bei":
        memory = await get_player_memory(player_id)

    # 5. 内容安全检查
    safety_result = await check_content_safety(message)
    if safety_result.has_crisis_signal:
        # 触发危机处理流程
        return await handle_crisis(player_id, npc_id, message)

    # 6. 情绪分析（仅小北）
    emotion = None
    if npc_id == "bei":
        emotion = await analyze_emotion(message)

    # 7. 构建Prompt
    prompt = build_npc_prompt(
        npc_config=npc_config,
        player_context=player_context,
        dialogue_history=dialogue_history,
        memory=memory,
        emotion=emotion,
        user_message=message,
    )

    # 8. 调用LLM
    response_stream = await call_llm_stream(
        model=select_model_for_npc(npc_id),
        prompt=prompt,
    )

    # 9. 流式返回响应
    full_response = ""
    async for chunk in response_stream:
        full_response += chunk
        yield {"type": "stream", "content": chunk}

    # 10. 保存对话历史
    await save_dialogue(player_id, npc_id, message, full_response)

    # 11. 更新记忆（仅小北）
    if npc_id == "bei":
        await update_player_memory(player_id, message, full_response, emotion)

    # 12. 返回完成信号
    yield {"type": "complete", "content": full_response}
```
