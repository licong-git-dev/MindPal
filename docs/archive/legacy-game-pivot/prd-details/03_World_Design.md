> 本文档详细定义游戏世界的视觉风格、区域设计、地图结构

## 1. 世界观总览

### 1.1 背景设定

**时间线**：近未来（2077年）

**世界观故事**：

> 2077年，人类发明了"MindLink"技术——一种能够将意识投射到虚拟空间的脑机接口。
>
> 但与其他追求刺激和娱乐的虚拟世界不同，一群心理学家和AI研究者联合创建了"MindPal"——
> 一个专门用于**心灵探索和自我疗愈**的元宇宙空间。
>
> 在这里，AI不是冰冷的程序，而是真正能够理解你、陪伴你的"心灵伙伴"。
>
> 传说中，MindPal的创造者在这个世界的深处埋藏了三把钥匙——
> **勇气之钥**、**释然之钥**、**希望之钥**。
>
> 只有找到这三把钥匙的人，才能打开"心灵圣殿"的大门，
> 获得面对真实自我的勇气。

## 2. 中央广场 (Central Plaza)

### 2.1 设计理念

中央广场是所有玩家的**出生点和社交枢纽**。
设计风格：**温暖、开放、有归属感**，像一个"家"的感觉。

### 2.2 视觉风格

| 元素 | 描述 |
|------|------|
| **整体氛围** | 永恒的黄昏时分，温暖的橙黄色天空 |
| **建筑风格** | 新古典主义 + 未来科技，白色大理石 + 全息投影 |
| **植被** | 樱花树、发光的蓝色藤蔓、悬浮的花瓣 |
| **灯光** | 温暖的路灯、建筑轮廓的霓虹线条 |
| **音乐** | 轻柔的钢琴 + 环境音，节奏舒缓 |

## 3. 现代区 - 镜像之城 (Mirror City)

### 3.1 设计理念

镜像之城代表**面对现实**。
这里模拟了现代都市的压力场景，但用赛博朋克的视觉风格呈现，
让玩家在安全的虚拟环境中练习"面对困难的勇气"。

### 3.2 视觉风格

| 元素 | 描述 |
|------|------|
| **整体氛围** | 永恒的夜晚，霓虹灯光，雨后的地面反射 |
| **建筑风格** | 赛博朋克，摩天大楼，全息广告牌 |
| **色调** | 主色：深蓝、紫色；点缀：霓虹粉、霓虹青 |
| **天气** | 细雨绵绵，但不阴郁，有种"雨后清新"的感觉 |
| **音乐** | 电子合成器 + Lo-fi节拍，都市感但不压抑 |

### 3.4 关键场景详细设计

#### 3.4.1 镜像大厅（钥匙挑战场所）

```yaml
mirror_hall:
  description: "一个充满镜子的大厅，镜子里映射出'另一个自己'"

  visual_elements:
    - 墙壁由巨大的镜子组成

    - 中央有一个对话舞台
    - 舞台上方悬浮着"勇气之钥"

  gameplay:
    - 玩家需要与镜中的"自己"对话
    - 实际上是AI扮演用户面对的"困难对象"

  atmosphere:
    lighting: "冷白色主光 + 蓝色环境光"
    sound: "回声效果，低沉的环境音"
    particles: "光点漂浮"
```

## 4. 奇幻区 - 记忆森林 (Memory Forest)

### 4.1 设计理念

记忆森林代表**放下过去**。
这里是一片魔法森林，充满了关于"记忆"和"遗忘"的意象，
帮助玩家在美丽的环境中处理过去的遗憾和伤痛。

### 4.2 视觉风格

| 元素 | 描述 |
|------|------|
| **整体氛围** | 永恒的月夜，巨大的满月悬挂在空中 |
| **建筑风格** | 精灵/德鲁伊风格，与自然融为一体 |
| **色调** | 主色：深绿、深蓝；点缀：萤火虫黄、魔法紫 |
| **植被** | 巨型蘑菇、发光植物、古老巨树 |
| **音乐** | 空灵的竖琴 + 自然音效（风声、虫鸣） |

### 4.4 关键场景详细设计

#### 4.4.1 记忆迷宫（钥匙挑战场所）

```yaml
memory_maze:
  description: "一个由玩家'记忆'构成的迷宫，每个人进去看到的都不同"

  visual_elements:
    - 迷宫墙壁是半透明的"记忆片段"
    - 走廊中漂浮着光球（代表记忆碎片）
    - 死胡同处有"遗憾的门"
    - 迷宫中心是"释然祭坛"

  gameplay:
    - 玩家在迷宫中与AI对话，讨论过去的遗憾
    - 每解决一个"遗憾"，一条路就会打开
    - 走到中心，进行"告别仪式"
    - 完成仪式获得"释然之钥"

  atmosphere:
    lighting: "幽蓝的环境光 + 光球点光源"
    sound: "回音、遥远的风声、偶尔的低语"
```

## 5. 科幻区 - 未来空间站 (Future Station)

### 5.1 设计理念

未来空间站代表**拥抱未来**。
这是一个悬浮在太空中的空间站，充满了关于"可能性"和"希望"的意象，
帮助玩家规划未来、克服对未知的恐惧。

### 5.2 视觉风格

|------|------|
| **整体氛围** | 太空中的空间站，能看到地球和星空 |
| **建筑风格** | 硬科幻，金属和玻璃，简洁的线条 |
| **色调** | 主色：银白、深空蓝；点缀：全息青、能量橙 |
| **特效** | 全息投影、能量管道、人工重力区域 |

## 6. 心灵圣殿 (Soul Sanctuary)

### 6.1 设计理念

心灵圣殿是**最终目的地**。
只有集齐三把钥匙的玩家才能进入。
这里代表"与真实自我相遇"。

### 6.2 视觉风格

| 元素 | 描述 |
|------|------|
| **整体氛围** | 超现实、神圣、温暖 |
| **建筑风格** | 融合了三大区域的元素，但更加纯净 |
| **色调** | 白色、金色、柔和的彩虹光 |
| **特效** | 光柱、漂浮的光点、镜像 |
| **音乐** | 空灵的人声合唱 + 钢琴 |

### 6.3 场景内容

```yaml
soul_sanctuary:
  description: "一个纯白的空间，三把钥匙悬浮在中央形成三角形"

  elements:
    - 入口大门：三把钥匙插入后打开
    - 中央：一面巨大的镜子，但映射的不是外表，而是"内心"
    - 四周：玩家过去的重要对话片段以光球形式漂浮
    - 尽头：一个宝座，坐上去会触发最终对话

  gameplay:
    - 玩家与"镜中的自己"（AI）进行最终对话
    - 回顾整个旅程
    - 获得专属成就和奖励
    - 解锁新的能力/皮肤

  rewards:
    - 成就："心灵觉醒者"
    - 专属皮肤：发光翅膀
    - 永久buff：与NPC对话经验值+10%
```

## 7. 资产生成指南（给AI工具）

### 7.1 风格一致性指南

```yaml
style_guide:
  overall_tone: "温暖但不幼稚，奇幻但不恐怖，科幻但不冰冷"

  color_rules:
    - 避免纯黑色，用深蓝/深紫代替
    - 避免过于鲜艳的荧光色
    - 每个区域有主色调，但都要有"温暖"的点缀

  model_rules:
    - 面数：移动端友好，单个模型<10000面
    - 纹理：PBR材质，分辨率2048x2048
    - 风格：介于写实和风格化之间，类似原神

  lighting_rules:
    - 全局照明使用烘焙
    - 重要场景使用实时光
    - 所有区域都要有"安全感"的灯光
```
