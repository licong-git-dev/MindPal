# MindPal PRD - 01 产品愿景

> 本文档定义产品的核心愿景、价值主张和设计原则

---

## 1. 产品愿景

### 1.1 一句话愿景

**"在虚拟世界中找到真实的自己"**

MindPal是一个让人们可以安全探索内心、获得情绪支持、建立有意义连接的元宇宙空间。

### 1.2 长期愿景（3-5年）

成为全球领先的**情绪健康元宇宙平台**，让每个人都有一个：
- 随时可以倾诉的AI朋友
- 安全探索自我的虚拟空间
- 志同道合的真人社区

### 1.3 产品定位矩阵

```
                    高情感价值
                        │
                        │
         ┌──────────────┼──────────────┐
         │              │              │
         │   心理App    │   MindPal    │
         │              │     ★        │
         │              │              │
低游戏性 ├──────────────┼──────────────┤ 高游戏性
         │              │              │
         │   聊天机器人 │   VRChat/    │
         │              │   原神       │
         │              │              │
         └──────────────┼──────────────┘
                        │
                    低情感价值
```

**MindPal的独特位置：高情感价值 + 高游戏性**

## 2. 核心价值主张

### 2.1 对用户的价值

| 价值维度 | 具体体现 | 用户获得 |
|---------|---------|---------|
| **情绪疗愈** | AI朋友深度对话、情绪识别、共情回应 | 被理解、被接纳的感觉 |
| **安全空间** | 虚拟身份、匿名社交、无评判环境 | 可以做真实自己的自由 |
| **游戏乐趣** | 探索、任务、收集、成就 | 即时满足感和成就感 |
| **社交连接** | 多人在线、组队、好友系统 | 归属感和不孤独 |
| **自我成长** | 反思引导、问题解决、未来规划 | 对生活的新视角 |

### 2.2 情绪疗愈的设计哲学

**不是治疗，是陪伴**

| 我们做的 | 我们不做的 |
|---------|-----------|
| 倾听和共情 | 诊断和治疗 |
| 提供视角和建议 | 给出医学处方 |
| 营造正能量环境 | 深挖创伤记忆 |
| 引导自我反思 | 强迫用户面对 |
| 在危机时引导至专业资源 | 替代专业心理咨询 |

### 2.3 核心体验公式

```
MindPal体验 = 游戏化外壳 × AI情感智能 × 社交连接 × 安全环境
```

## 3. 世界观概述

### 3.1 背景故事

> 在不远的未来，人类发明了一种叫做"MindPal"的虚拟空间。
>
> 这不是一个逃避现实的地方，而是一个**面对自己**的地方。
>
> 在这里，每个人都可以找到一个"懂你"的AI伙伴，
> 可以在三个奇妙的世界中探索、冒险、成长。
>
> 传说中，完成三个世界的试炼，找到三把钥匙，
> 就能打开"心灵圣殿"的大门——
> 在那里，你将遇见最真实的自己。

### 3.2 三大区域概述

| 区域 | 主题 | 情绪疗愈方向 | 视觉风格 |
|------|------|-------------|---------|
| **现代区（Mirror City）** | 面对现实 | 勇气、沟通、人际关系 | 赛博朋克都市 |
| **奇幻区（Memory Forest）** | 放下过去 | 释然、原谅、告别 | 魔法森林 |
| **科幻区（Future Station）** | 拥抱未来 | 希望、规划、方向 | 太空站 |

### 3.3 三把钥匙

| 钥匙 | 所在区域 | 获取方式 | 象征意义 |
|------|---------|---------|---------|
| **勇气之钥** | 现代区 | 完成"镜像对话"挑战 | 敢于面对困难 |
| **释然之钥** | 奇幻区 | 完成"记忆迷宫"挑战 | 学会放下过去 |
| **希望之钥** | 科幻区 | 完成"未来建筑师"挑战 | 相信美好未来 |

**集齐三把钥匙 → 解锁"心灵圣殿" → 获得专属成就和奖励**

## 4. 设计原则

### 4.1 情感设计原则

| 原则 | 具体要求 |
|------|---------|
| **安全第一** | 任何功能设计都要考虑用户的心理安全 |
| **正向引导** | 始终营造积极、温暖、有希望的氛围 |
| **尊重边界** | 不强迫用户分享，不过度追问 |
| **记住用户** | AI要记住用户说过的话，体现被重视感 |
| **提供出口** | 在检测到危机信号时，引导至专业资源 |

### 4.2 游戏设计原则

| 原则 | 具体要求 |
|------|---------|
| **低门槛** | 新手也能轻松上手，有完整的引导教程 |
| **高上限** | 有足够的深度内容供老玩家探索 |
| **即时反馈** | 每个行为都有视觉/音效反馈 |
| **渐进解锁** | 内容逐步解锁，保持新鲜感 |
| **不强迫氪金** | 付费内容为装饰和便利，非核心体验 |

### 4.3 技术设计原则

| 原则 | 具体要求 |
|------|---------|
| **模块化** | 每个功能独立模块，便于维护和迭代 |
| **可扩展** | 架构支持后续添加新区域、新NPC、新玩法 |
| **稳定性** | 多人系统要有容错机制，单点故障不影响全局 |
| **可观测** | 完整的日志和监控，便于排查问题 |
| **代码清晰** | 命名规范、注释完整，便于AI理解和修改 |

## 5. 用户旅程

### 5.1 新用户首次体验（前30分钟）

```
1. 启动游戏
   └── 品牌动画 + 背景音乐

2. 创建账号
   └── 邮箱/手机号注册
   └── 设置用户名和密码

3. 创建Avatar
   └── 选择基础形象（性别、肤色、发型）
   └── 简单自定义（服装、配饰）
   └── AI提示："你想让自己看起来像什么样的人？"

4. 进入世界
   └── 出生点：中央广场
   └── 导游NPC「艾拉」主动走来打招呼

5. 新手引导（由艾拉引导）
   └── 学会移动和视角控制
   └── 学会与NPC对话
   └── 认识朋友NPC「小北」和宠物「芝麻」
   └── 了解三个区域和三把钥匙的故事

6. 第一次深度对话
   └── 小北询问："你今天过得怎么样？"
   └── 玩家体验AI对话的魅力
   └── 小北记住玩家分享的内容

7. 领取第一个任务
   └── 任务发布者「时守」出现
   └── 发布简单的探索任务
   └── 完成后获得游戏货币奖励

8. 引导结束
   └── 艾拉："现在你可以自由探索了！"
   └── 显示三个区域的入口
   └── 玩家开始自由游玩
```

### 5.2 核心游玩循环

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   探索世界 ──→ 遇到NPC ──→ 对话/任务 ──→ 获得奖励      │
│       ↑                                      │          │
│       │                                      ↓          │
│       └──────────── 解锁新内容 ←─────────────┘          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 5.3 情绪疗愈循环

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   感到困扰 ──→ 找小北倾诉 ──→ 被倾听和理解 ──→ 获得建议 │
│       ↑                                        │        │
│       │                                        ↓        │
│       └──────── 感觉好一些，继续游玩 ←─────────┘        │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## 6. 商业模式

### 6.1 收入来源

| 来源 | 占比预期 | 具体内容 |
|------|---------|---------|
| **游戏货币充值** | 60% | 购买金币用于商店消费 |
| **高级会员订阅** | 30% | 每月订阅解锁更多功能 |
| **特殊皮肤/道具** | 10% | 限定版装饰直购 |

### 6.2 货币体系

| 货币 | 获取方式 | 用途 |
|------|---------|------|
| **金币（免费）** | 每日登录、完成任务、成就奖励 | 基础道具、普通装饰 |
| **钻石（付费）** | 充值购买 | 高级道具、稀有装饰、加速道具 |

### 6.3 定价策略

| 项目 | 价格 | 说明 |
|------|------|------|
| 基础游戏 | 免费 | 核心体验完全免费 |
| 金币包（小） | ¥6 | 100金币 |
| 金币包（中） | ¥30 | 600金币 |
| 金币包（大） | ¥98 | 2500金币 |
| 高级会员 | ¥25/月 | 每日金币翻倍 + 专属皮肤 + 优先匹配 |
