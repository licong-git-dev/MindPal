## 1. 设计原则

### 1.1 核心设计理念

| 原则 | 说明 | 实践 |
|------|------|------|
| **沉浸优先** | 界面不破坏3D世界沉浸感 | 半透明、淡入淡出、贴边设计 |
| **信息克制** | 只显示当前必要信息 | 自动隐藏、悬停显示 |
| **情感化** | 界面传递温暖感 | 圆角、暖色、微动效 |
| **无障碍** | 考虑不同用户群体 | 字体可调、色盲模式 |

### 1.2 色彩系统

```
主色调（温暖科幻）
├── Primary: #6C5CE7 (梦幻紫)
├── Secondary: #00CEC9 (科技青)
├── Accent: #FD79A8 (温暖粉)
├── Background: #1A1A2E (深空蓝)
└── Surface: #16213E (浅夜蓝)

功能色
├── Success: #00B894
├── Warning: #FDCB6E
├── Error: #E17055
└── Info: #74B9FF

文字色
├── Primary Text: #FFFFFF (100%)
├── Secondary Text: #FFFFFF (70%)
├── Disabled Text: #FFFFFF (40%)
└── Hint Text: #FFFFFF (50%)
```

### 1.3 字体规范

```
中文字体: "思源黑体", "Noto Sans SC"
英文字体: "Rajdhani", "Roboto"
数字字体: "Orbitron" (科幻感)

字号层级:
├── H1: 32px (页面标题)
├── H2: 24px (区块标题)
├── H3: 18px (卡片标题)
├── Body: 16px (正文)
├── Caption: 14px (说明文字)
└── Small: 12px (辅助信息)
```

## 6. 动效规范

### 6.1 过渡动画

| 场景 | 动画类型 | 时长 | 缓动函数 |
|------|----------|------|----------|
| 面板打开 | 淡入+缩放 | 200ms | ease-out |
| 面板关闭 | 淡出+缩放 | 150ms | ease-in |
| 页面切换 | 滑动 | 300ms | ease-in-out |
| 按钮悬停 | 发光+缩放 | 100ms | linear |
| 通知弹出 | 从上滑入 | 250ms | spring |
| 获得物品 | 弹跳+闪光 | 400ms | bounce |

### 6.2 微交互

```
按钮点击:
1. 按下: 缩放至0.95, 50ms
2. 释放: 缩放至1.05, 100ms
3. 恢复: 缩放至1.0, 100ms
4. 添加涟漪扩散效果

输入框聚焦:
1. 边框颜色渐变为主色
2. 添加柔和发光
3. 占位符上浮为标签

加载状态:
1. 旋转加载圈 (紫色渐变)
2. 脉动效果
3. 进度百分比文字
```

## 7. 响应式适配

### 7.1 分辨率适配

| 分辨率 | UI缩放 | HUD调整 |
|--------|--------|---------|
| 1920x1080 | 100% | 标准布局 |
| 2560x1440 | 125% | 增大边距 |
| 3840x2160 | 150% | 4K优化 |
| 1366x768 | 85% | 紧凑布局 |

### 7.2 字体自适应

```gdscript
func get_scaled_font_size(base_size: int) -> int:
    var screen_height = DisplayServer.window_get_size().y
    var scale = screen_height / 1080.0
    return int(base_size * clamp(scale, 0.8, 1.5))
```

## 8. 无障碍设计

### 8.1 色盲模式

```
普通模式 → 红绿色盲模式
#E17055 (红) → #0077BB (蓝)
#00B894 (绿) → #EE7733 (橙)

普通模式 → 蓝黄色盲模式
#74B9FF (蓝) → #CC3311 (红)
#FDCB6E (黄) → #009988 (青)
```

### 8.2 辅助功能

- **字体大小**: 支持80%-150%调节
- **高对比度**: 可选高对比度主题
- **屏幕阅读器**: 所有UI元素添加aria标签
- **键盘导航**: 完整Tab键支持
- **字幕**: 所有语音内容显示字幕
